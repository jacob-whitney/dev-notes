# Steps to Setup a Server

## Windows PC

1. Activate OpenSSH
2. Install Ubuntu WSL
   1. Create Linux User Accounts
      1. Configure default user for WSL
3. Configure git
4. Configure .bashrc
5. Configure .vimrc

### Create Additional Virtual Machines

1. Install VirtualBox

## Virtual Machine

1. Activate OpenSSH or activate SSH server?
2. Install Ubuntu
3. Install Apache
4. Install PHP
5. Install MySQL
6. Configure git
7. Configure .bashrc
8. Configure .vimrc
